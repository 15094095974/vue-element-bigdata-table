# vue-element-bigdata-table

> Vue2 elementUI table 组件扩展，大量数据表格。

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run dist
```
## Feature

采用虚拟渲染方案，解决大数据量DOM渲染性能瓶颈。
