# vue-element-bigdata-table

> Vue2 elementUI table 组件扩展，大量数据表格。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
## Feature

采用虚拟渲染方案，解决大数据量DOM渲染性能瓶颈。
